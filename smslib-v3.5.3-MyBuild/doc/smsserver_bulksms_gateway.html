SMSServer - BULKSMS gateway

This gateway is used for the BULKSMS HTTP interface.

Configuration File

This interface requires the following settings in SMSServer's configuration file:

OptionDescription
usernameBULKSMS Username.
passwordBULKSMS Password.
outboundyes if the gateway should be treaded as an outbound gateway, otherwise no.
regionThe BULKSMS region servers you wish to use. Should be one of the following: UNITEDKINGDOM, SOUTHAFRICA, SPAIN, USA, GERMANY. Leave the setting empty to use the INTERNATIONAL setting.

Example:

gateway.0=myBulkSms, BulkSmsHttp
myBulkSms.username=username
myBulkSms.password=password
myBulkSms.outbound=yes
myBulkSms.region=USA
