Compatible Bulk SMS Operators

SMSLib currently supports BULKSMS (all regions) and Clickatell.

Please note that the above mentioned bulk sms operators are supported only for outbound messaging. SMSLib implements an HTTP/HTTPS interface with them.

Incompatible GSM Modems/Phones

The following models are reported to be incompatible:

  • Nokia: All Symbian models (they partially work or don't work at all).
  • Nokia: 6131.
  • Blackberry 8700g.
  • Nokia 5610 XpressMusic.
  • Motorola G18 GSM Modem.
